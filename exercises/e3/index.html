Harjoitus 3 - Pull Request

Tämä tiedosto on tehty uuteen kehityshaaraan (branch)

Tästä branchista on tehty pull/merge request, eli pyyntö yhdistää branchin sisältö päähaaraan main/master.

  • Tallenna paikallisella koneella tehdyt muutokset gittiin.
  • Toteuta yhdistäminen GitHubin Pull requests välilehdellä.
  • Synkronoi paikallinen kansio komennolla git pull
  • Nyt voi taas commitoida uuden version main/master haaran päälle
